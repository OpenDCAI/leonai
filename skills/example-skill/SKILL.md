# Example Skill

## Overview

This is an example skill that demonstrates how to create specialized capabilities for Leon.

## Instructions

When this skill is loaded, you gain access to specialized knowledge about a particular domain.

### Step 1: Understand the Context

Review the user's request and determine if this skill's domain knowledge is relevant.

### Step 2: Apply Domain Knowledge

Use the specialized instructions and context provided in this skill to complete the task.

### Step 3: Provide Accurate Guidance

After applying the skill's knowledge, complete the user's request with domain-specific expertise.

## Example Usage

This skill can be used for tasks related to its specific domain. Load it when you need specialized knowledge that isn't part of your general capabilities.
